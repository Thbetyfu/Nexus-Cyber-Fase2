# 🔒 Skill: PQC (Post-Quantum Cryptography)

## Objective
Mengintegrasikan kriptografi tahan-komputer-kuantum untuk melindungi data sensitif.

## Workflow
1. **Lattice-based Cryptography**: Implementasi ML-KEM (Kyber).
2. **Key Encapsulation**: Proteksi terhadap ancaman "Harvest Now, Decrypt Later".

## Constraints
* Dilarang menggunakan RSA, ECC murni, atau AES standar tanpa wrapper PQC untuk data sensitif.
