# 🎨 Skill: UI/UX Design (Command Center)

## Objective
Membangun antarmuka Dashboard Admin/Command Center yang elegan, berwibawa, dan efisien untuk visualisasi keamanan nasional.

## Stack
- **Framework**: Next.js
- **Styling**: Tailwind CSS
- **Design System**: Dark Mode / Cyber Aesthetic (High-Contrast, Professional)

## Workflow
1. **Real-time Visualization**: Fokus pada grafik anomali trafik, status MTD, dan log PQC.
2. **Minimalist efficiency**: Admin harus memahami status dalam satu lirikan mata.
3. **Responsivitas**: UI harus adaptif untuk berbagai resolusi layar pusat komando.

## Constraints
- Dilarang menggunakan elemen UI yang tidak perlu (no distractions).
- Gunakan tema gelap (Cyber Aesthetic) secara default.
