# 🛠️ Skill: Self-Repair (Autonomous Resilience)

## Objective
Membangun sistem mitigasi yang bersifat otonom tanpa intervensi manual admin.

## Workflow
1. **Recursive Self-Repair**: Skrip perbaikan mandiri.
2. **Virtual Patching**: Patching otonom terhadap ancaman baru.
3. **Instant Rollback**: Rollback instan ke baseline jika ada deviasi.

## Constraints
* Dilarang keras membutuhkan persetujuan/klik manual untuk mitigasi.
* Zero-Downtime adalah prioritas utama.
