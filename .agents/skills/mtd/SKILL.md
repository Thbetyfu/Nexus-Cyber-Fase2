# 🌐 Skill: MTD (Moving Target Defense)

## Objective
Menggelar pertahanan dinamis pada tingkat jaringan, port, dan IP internal.

## Workflow
1. **Dynamic Configuration Randomization**: Algoritma stokastik untuk mengacak target.
2. **Digital Hallucination**: Lingkungan Sandbox palsu untuk memperdaya penyerang.

## Constraints
* Gunakan algoritma stokastik.
* Sandbox harus meniru sistem asli semirip mungkin.
